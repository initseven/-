### 安装python3 pip3

- ```python3 -m pip install --upgrade --force-reinstall pip```

- 添加pip3所在目录到环境变量

### 安装pipenv
- ```pip3 install pipenv```

### 配置pipenv
- ```进入项目目录E:\init_project\flask-vuejs-template>```

- 创建虚拟环境，安装依赖```pipenv install --dev```

- 激活虚拟环境并进入环境```pipenv shell```

- 退出虚拟环境```exit```
